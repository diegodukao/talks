## Ionic / PhoneGap (Cordova) - ## Entre nativo e webaapp
## Web App? - ### Uma página web otimizada para tablet e celulares
## Vantagens (Web App) - ### Você não precisa aprender a programar para aquele SO específico - ### HTML + CSS + Javascript - ### Pode ser apenas uma versão mobile do seu site que já está no ar
## Desvantagens (Web App) - ### Acesso limitado aos recursos do dispositivo - ### Mais lenta que apps nativas
